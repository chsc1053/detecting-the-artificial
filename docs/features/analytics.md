## Scope

- **Single admin panel** (auth-protected): includes an **analytics section** where analytics are **automated** from the stored data (responses, participants, trials, stimuli).
- Examples: accuracy and confidence by study/trial, calibration, breakdowns by modality or model, demographics vs. accuracy; export for further analysis.
- No separate “analytics product”; the researcher uses this panel to run studies and view/compute analytics in one place.
